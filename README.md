# Image Compressor

A simple and fast Image Compressor built using HTML, CSS, and JavaScript. This web application allows users to upload an image, preview it, compress it by adjusting quality, and download the compressed version.

## Features

- Upload images from your device
- Preview selected image
- Display original image size
- Compress image with custom quality settings
- Display compressed image size
- Download compressed image
- Responsive and user-friendly interface
- No server required (works completely in the browser)

## Technologies Used

- HTML5
- CSS3
- JavaScript (Canvas API)

## How to Use

1. Open the application in your browser.
2. Click **Choose Image** and select an image.
3. Adjust the compression quality.
4. Click **Compress Image**.
5. View the compressed image and size.
6. Click **Download** to save the compressed image.
